End-to-end system flow

Signals enter privately, Major routes, employees execute, approvals gate risk, and outcomes return to the dashboard.

3
Specialist route
Riley, Mina, Reese, Tilly, Dash, Drew, and Logan pick up function-specific work.
4
Skills + tools
M365 tools, MCP servers, WorkIQ, web tools, and .md skill contracts guide execution.

Approval inbox
Your decisions, plus proactive outward items an employee isn't yet trusted to send on its own, wait for you. Anything you approve is carried out for real.

Private results
Sent replies, drafts, docs, decks, prep briefs, summaries, and links return to Results.
7
Evidence + logs
Activity log records operations; Work & Impact Ledger records brief completed work plus highlighted outcomes.

Runtime architecture layers

The app is local-first: the browser reads the local API; automations and tools write back through explicit records.

Python API
