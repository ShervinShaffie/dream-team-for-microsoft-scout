# Release notes — v4.0.0 (first public release)

**The Dream Team for Microsoft Scout** — your own team of eight AI digital employees, running locally on Microsoft Scout. This is the first public, GitHub-distributed edition.

## 🪟 Windows only
The installer and local app run on **Windows 10/11**. The setup scripts are Windows-specific and will not run on macOS or Linux.

## What's in this release
- A complete, eight-employee team that works for **any** Microsoft Scout user signed into their own Microsoft 365 — no corporate sign-in required for the core experience.
- A guided setup wizard that **auto-detects** a Microsoft sign-in and tailors the install: Microsoft employees are offered optional internal depth skills; everyone else gets the full baseline team with zero internal references.
- Scout-native fallbacks so every employee does its job even without the optional depth skills.
- A portable document folder that resolves your OneDrive `Scout` folder no matter how OneDrive is named, with a local fallback.

## Get started
1. Download `dream-team-for-microsoft-scout-v4.0.0.zip` below.
2. Unzip anywhere.
3. Double-click **`START HERE.cmd`** and follow the 3 steps.

Full instructions: see the [README](../../blob/main/README.md).

## Privacy & safety
Runs entirely on `127.0.0.1`, stores to a local database, and contains **no personal data** (enforced by a clean-room verifier). The team drafts — it never sends email, Teams, or calendar responses to others without your explicit approval.
