# What's new in 4.0.2

This release fixes the most common setup problem and fills out the project history.

- Fixed a confusing setup error. If you started setup from inside the downloaded zip without extracting it first, you used to see a red error about a missing install file. START HERE.cmd now catches this and tells you, in plain words, to extract the zip first and try again. Check Setup.cmd does the same.
- Added a full changelog so you can see how the Dream Team has grown across every release, with a short summary of what it does today at the top.
- Cleaned up the README so it reads more plainly.

No change to how the app or the team works.
